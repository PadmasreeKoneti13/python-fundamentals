# Python Fundamentals

This repository contains my practice problems as I master the building blocks of Python.

## Topics Covered
* **Conditionals:** Logic gates, `if/elif/else` statements, and decision-making.
* **Loops:** Using `while` and `for` loops for repetition and input validation.
* **Functions:** Modular code using parameters and `return` values.
* **Collections:** Working with Lists, Tuples, Sets, and Dictionaries.
* **OOP:** Introduction to Object-Oriented Programming and Classes.

## Featured Programs
* **Calculator:** A terminal-based program with nested loops and error checking.
* **Logic Scripts:** Problems covering Fibonacci, Factorials, and String manipulation.

## Environment
* **Language:** Python 3
* **IDE:** PyCharm
